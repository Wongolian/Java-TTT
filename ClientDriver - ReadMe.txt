Run the ServerDriver.jar file first.

Run the ClientDriver.jar file.

Input desired username, ip address and port you wish to connect with. Press connect.
(Port should be 5555 for example as it is hardcoded in the server)

Wait for a second user to join.

Once second user has join, the server will randomly select a user to go first.

Commence playing Tic tac toe.

Type a message in the message box if you wish to chat with another user. Press Send to send it to your opponent.
